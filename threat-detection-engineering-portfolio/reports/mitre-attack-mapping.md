## Validated Detection Summary

### DET-001: Sigma PowerShell Encoded Command

The Sigma detection identified PowerShell execution using EncodedCommand. The encoded command was safely generated in the lab and decoded to:

Start-Sleep -Seconds 30

This maps to:

T1059.001 - PowerShell

### DET-002: Wazuh SSH Invalid User Password Guessing

The Wazuh custom rule detected an SSH failed login attempt for an invalid user. The event was tested with wazuh-logtest inside a running Wazuh manager container.

This maps to:

T1110.001 - Password Guessing

### DET-003: Falco Shell Spawned In Container

The Falco custom rule detected shell execution inside a Kubernetes container in the detection-lab namespace. The alert was generated by the running Falco DaemonSet.

This maps to:

T1611 - Escape to Host
