\# Snort Lab



This project documents my Snort lab setup, configuration, and testing.



\## Contents



\- `configs/` - configuration files

\- `rules/` - custom or test rules

\- `screenshots/` - setup and testing screenshots

\- `notes/` - install notes and observations



\## Summary



In this lab, I installed and configured Snort, validated the installation, and prepared the environment for traffic inspection and alert generation.
